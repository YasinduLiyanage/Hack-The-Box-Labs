1. FTP protocol

FTP (File Transfer Protocol) is based on a client-server model, which allows the transfer of files between them.

FTP has a security issue. It is default FTP traffic is unencrypted (plaintext), so it can be faced into Mitm attacks. FTPS(SSL/TLS) or SFTP(SSH tunnelling) are the secured version of FTP.
